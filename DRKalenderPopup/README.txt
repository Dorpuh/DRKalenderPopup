DRKalenderPopup
Autor: Abgedropst
Gilde: Darkness Rising

Funktion:
- Prüft nach jedem Login auf Gildenereignisse im Kalender.
- Zeigt das Popup bei jedem Charakter erneut an, solange auf das Ereignis noch nicht reagiert wurde.
- Erst ein Klick auf OK markiert das Ereignis als erledigt.
- Klick auf OK öffnet den Blizzard-Kalender.
- Größeres Gildenlogo im Popup.

Slash-Befehle:
/drpopup test   -> zeigt Test-Popup
/drpopup scan   -> startet erneute Kalendersuche
/drpopup reset  -> setzt gespeicherte Ereignisse zurück
