🇬🇧 DRKalenderPopup – Smart Calendar Event Reminder
Overview
DRKalenderPopup is a World of Warcraft addon designed to ensure you never miss important calendar events again.

Originally created for the guild Darkness Rising, the addon now supports all relevant calendar events, not just guild events.

Whenever a new or pending event is detected, a large, stylized popup appears with all important details.

✨ Features
📅 Smart Event Detection
Detects new and unconfirmed calendar events

Works with:

Guild events

Player-created events

Community events

🪟 Immersive Popup UI
Large, guild-themed popup window

Displays:

Event title

Event date

Description

Clean and responsive layout (UI scale supported)

✔️ Event Completion System
Mark events as completed via checkbox

Completed events:

Will not reappear

Persist across relogs

🔁 Event Navigation
If multiple events are pending:

Navigate using left/right arrows

View events one by one

⏳ Snooze System
"Later" and "X" minimize the popup

Popup becomes a movable mini-icon

Click the icon to restore the popup

🧲 Movable Reminder Icon
Freely draggable on screen

Position is saved

Subtle glow animation for visibility

🗺️ Minimap Button
Custom minimap button with your addon icon

Left Click → open options menu

Right Click → open calendar

Drag to reposition around minimap

⚙️ Options Menu
Quick access to all addon functions:

Test popup

Manual calendar scan

Reset events

Clear data

Open calendar

Toggle minimap button

⚔️ Combat & Instance Safe
No popup during:

Combat

Dungeons / Raids / PvP / Scenarios

Automatically resumes afterward

🇩🇪 DRKalenderPopup – Intelligente Kalender-Erinnerung
Übersicht
DRKalenderPopup ist ein World of Warcraft Addon, das sicherstellt, dass du keine wichtigen Kalenderereignisse mehr verpasst.

Ursprünglich für die Gilde Darkness Rising entwickelt, unterstützt das Addon inzwischen alle relevanten Kalenderereignisse, nicht nur Gilden-Events.

Sobald ein neues oder noch offenes Ereignis erkannt wird, erscheint ein großes, stilisiertes Popup mit allen wichtigen Informationen.

✨ Features
📅 Intelligente Ereigniserkennung
Erkennt neue und unbestätigte Kalenderereignisse

Funktioniert mit:

Gildenereignissen

eigenen (Spieler-)Events

Community-Events

🪟 Stilvolles Popup
Großes Popup im Gilden-Design

Zeigt:

Eventname

Datum

Beschreibung

Responsives Layout (UI-Skalierung wird berücksichtigt)

✔️ Ereignis-Abschluss
Events können über einen Haken als erledigt markiert werden

Erledigte Events:

erscheinen nicht erneut

bleiben auch nach Relog gespeichert

🔁 Event-Navigation
Bei mehreren offenen Events:

Navigation über linke/rechte Pfeile

Events einzeln durchblättern

⏳ Später-Funktion
„Später“ und „X“ minimieren das Popup

Popup wird zu einem verschiebbaren Icon

Klick auf das Icon stellt das Popup wieder her

🧲 Verschiebbares Reminder-Icon
Frei auf dem Bildschirm platzierbar

Position wird gespeichert

Leicht pulsierende Animation

🗺️ Minimap-Button
Eigener Button an der Minimap

Linksklick → Optionsmenü

Rechtsklick → Kalender öffnen

Verschiebbar entlang der Minimap

⚙️ Optionsmenü
Schneller Zugriff auf alle Funktionen:

Test Popup

Kalender manuell scannen

Events zurücksetzen

Daten löschen

Kalender öffnen

Minimap-Button ein-/ausblenden

⚔️ Kampf- & Instanzsicher
Kein Popup während:

Kampf

Dungeons / Raids / PvP / Szenarien

Wird danach automatisch fortgesetzt
